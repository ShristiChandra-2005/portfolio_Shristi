File Zipper

The project implements a file compression utility that reduces the size of files for easier storage or transmission. It uses Huffman Coding, which is a widely-used lossless data compression algorithm, to compress the data. The Greedy Algorithm approach is used in Huffman Coding to construct the most efficient code based on the frequency of characters in the file.

Cash Flow Minimizer

The Cash Flow Minimizer aims to simplify a network of debts among multiple parties by minimizing the number of transactions while ensuring that all debts are paid off. The project uses graph-based data structures to represent debts and employs a Heap data structure for efficient processing of transactions and balance adjustments.

Sudoku Solver

The Sudoku Solver is a Java-based program that solves Sudoku puzzles using the Backtracking Algorithm. The solver fills empty cells of a given 9x9 Sudoku grid while ensuring that all rules of Sudoku are satisfied: each row, column, and 3x3 subgrid must contain all digits from 1 to 9 without repetition.
